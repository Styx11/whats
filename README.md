## Install

```
npm install -g whats-this
```

## Usage

```
Usage: whats <word> [options]

Options:
  -v, --vers  output the current version
  -h, --help  output usage information

Examples:
  $ whats love
  $ whats 爱
```
**测试版本目前只支持中英文基本词语的互译**
